# 四、核心：ChatService（业务大脑）

```python
class ChatService:

    def handle(self, req):

        # 1. 意图识别（业务核心）
        intent = self.detect_intent(req.query)

        # 2. 路由到不同业务
        if intent == "policy":
            return self.policy_flow(req)

        elif intent == "service":
            return self.service_flow(req)

        elif intent == "health":
            return self.health_flow(req)

        else:
            return self.general_rag(req)
```

---

# 五、意图识别（不要复杂化）

你不需要训练模型，直接两种方式：

## 方案A（推荐，稳定）

```python
def detect_intent(self, query):

    if "补贴" in query or "政策" in query:
        return "policy"

    if "养老院" in query or "照顾" in query:
        return "service"

    if "吃什么" in query or "病" in query:
        return "health"

    return "general"
```

---

## 方案B（进阶，用Qwen分类）

```python
def detect_intent_llm(self, query):

    prompt = f"""
分类问题：
policy / service / health / general

问题：{query}
只输出类别
"""

    return qwen.generate(prompt)
```

# 六、四大业务流（重点）

---

## 1. 政策匹配流（最重要）

### 本质：结构化信息提取 + RAG

```python
def policy_flow(self, req):

    # 1. 提取用户条件（关键）
    profile = self.extract_user_profile(req.query)

    # 2. 调RAG（政策库）
    docs = rag.search(req.query, kb="policy")

    # 3. 拼装prompt（带条件）
    context = self.build_policy_context(profile, docs)

    answer = rag.generate(req.query, context)

    return self.wrap_response(answer, docs)
```

---

### 用户信息提取

```python
def extract_user_profile(self, query):

    prompt = f"""
提取信息：
年龄、户籍、是否有社保

问题：{query}

JSON输出
"""

    return qwen.generate(prompt)
```

---

## 2. 服务导办流

### 本质：DB筛选 + 引导

```python
def service_flow(self, req):

    # 1. 解析需求
    intent = self.parse_service_need(req.query)

    # 2. 查询数据库（不是RAG）
    results = service_repo.search(intent)

    # 3. 返回推荐
    return {
        "type": "service_recommend",
        "data": results
    }
```

---

## 3. 健康问答流（强限制）

```python
def health_flow(self, req):

    docs = rag.search(req.query, kb="health")

    answer = rag.generate(req.query, docs)

    # 强制免责声明
    answer += "\n\n（仅供科普参考，不构成医疗诊断）"

    return self.wrap_response(answer, docs)
```

---

## 4. 普通RAG

```python
def general_rag(self, req):

    docs = rag.search(req.query, kb="platform")

    answer = rag.generate(req.query, docs)

    return self.wrap_response(answer, docs)
```

# 七、响应统一结构（必须统一）

```python
def wrap_response(self, answer, docs):

    return {
        "answer": answer,
        "sources": [
            {
                "title": d["title"],
                "content": d["content"][:100]
            }
            for d in docs
        ]
    }
```

# 八、多轮对话（业务层实现）

你不需要改RAG，只做上下文拼接：

```python
def build_query(self, session_id, query):

    history = redis.get(session_id) or []

    return "\n".join(history[-3:] + [query])
```
